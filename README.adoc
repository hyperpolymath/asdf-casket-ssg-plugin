An {url-asdf}[asdf] plugin to manage https://github.com/hyperpolymath/casket-ssg[casket-ssg] versions.

== Installation

[source,bash]
----
asdf plugin add casket-ssg https://github.com/hyperpolymath/asdf-casket-ssg-plugin.git
----

== Usage

[source,bash]
----
# List all available versions
asdf list all casket-ssg

# Install a specific version
asdf install casket-ssg 1.1.0

# Set global default
asdf global casket-ssg 1.1.0

# Set local version for current project
asdf local casket-ssg 1.1.0
----
